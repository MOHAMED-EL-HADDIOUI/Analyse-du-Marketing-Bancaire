# Analyse du Marketing Bancaire
![Image](Bank.jpg)

## Description
Ce projet vise à analyser les données de marketing bancaire afin de comprendre les facteurs influençant l'adhésion aux dépôts à terme. En utilisant des techniques d'analyse de données et de machine learning, nous avons développé des modèles pour prédire la probabilité qu'un client souscrive à un dépôt à terme.

## Analyse des Données
Ce projet utilise un ensemble de données contenant des informations sur les clients bancaires, y compris l'âge, le solde du compte, et d'autres caractéristiques pertinentes. Une exploration des données a été réalisée pour visualiser les relations entre les caractéristiques des clients et l'adhésion aux dépôts.

## Modèles
Différents modèles de classification ont été appliqués, y compris :
- Random Forest
- AdaBoost
- Gradient Boosting
- Support Vector Classifier
- K-Nearest Neighbors
- Naive Bayes
- XGBoost (comme modèle de niveau 2)

## Résultats
Les résultats montrent que l'analyse des données et les modèles de machine learning permettent de mieux cibler les campagnes marketing, augmentant ainsi l'efficacité et le retour sur investissement.

## Conclusion
Les principales conclusions de cette analyse indiquent que le solde du compte et l'âge du client sont des facteurs significatifs influençant l'adhésion aux dépôts à terme. Des recommandations pour les campagnes marketing futures sont également fournies.
